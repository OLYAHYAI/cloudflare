My AI-Powered Workflow: Integration of MCP, Docker and N8N for Obsidian

üéØ Objective

Automate the organization of Obsidian notes using an AI accessible via API, orchestrated with Docker, MCP and N8N.

üßë‚Äçüíª Prerequisites

  • Docker installed
  • MCP (Modular Copilot Platform) deployed
  • N8N installed (via Docker or local)
  • Obsidian Vault accessible locally

üß™ Detailed Steps

1. üì¶ Deploy MCP with Docker

docker run -d \
  --name mcp \
  -p 5000:5000 \
  -v $(pwd)/mcp_data:/app/data \
  ghcr.io/modularcopilot/mcp:latest

MCP exposes an API on http://localhost:5000

2. üß† Prepare a prompt for the AI

Example prompt:

{
  "prompt": "Unify all my Obsidian notes in English. Translate French titles, group similar content, and create a thematic index."
}

3. ü§ñ Create an N8N workflow

Workflow steps:

  1. Trigger: manual or scheduled
  2. Read Files: read .md files from Obsidian vault
  3. HTTP Request: send content to MCP via POST
  4. URL: http://localhost:5000/api/prompt
  5. Body: JSON with prompt and notes content
  6. Write File: save response in a new .md file

4. ‚öôÔ∏è Example HTTP Request Configuration in N8N

  • Method: POST
  • URL: http://localhost:5000/api/prompt
  • Headers: Content-Type: application/json
  • Body:
{
  "prompt": "Organize my Obsidian notes by theme and language.",
  "notes": "{{ $json["content"] }}"
}

5. üìÅ Save in Obsidian

The generated file is saved in the Vault/Index/organization.md folder

You can link this file in your Obsidian home note

üß† Tips

  • Use tags #fr and #en to identify languages
  • Create a multilingual index note with links to each theme
  • You can enrich the prompt to include translations, groupings, or summaries

üîß Advanced Configuration

Docker Compose Setup

For a more robust setup, you can use Docker Compose:

version: '3.8'
services:
  mcp:
    image: ghcr.io/modularcopilot/mcp:latest
    container_name: mcp
    ports:
      - "5000:5000"
    volumes:
      - ./mcp_data:/app/data
    restart: unless-stopped
    
  n8n:
    image: n8nio/n8n:latest
    container_name: n8n
    ports:
      - "5678:5678"
    volumes:
      - ./n8n_data:/home/node/.n8n
    restart: unless-stopped

Automation Scheduling

You can schedule the workflow to run automatically:

  • Daily at 2 AM for note organization
  • Weekly for comprehensive indexing
  • On file changes using file watchers

Advanced Prompts

Example of an advanced prompt for better organization:

{
  "prompt": "Analyze all Obsidian notes and: 1) Group by technical domain (infrastructure, security, automation), 2) Create a hierarchical index with main topics and subtopics, 3) Translate all French content to English while preserving code blocks, 4) Generate summary tables for each category, 5) Suggest cross-references between related notes.",
  "options": {
    "preserve_code": true,
    "create_index": true,
    "translate_content": true,
    "generate_summaries": true
  }
}

  • Use batch processing for large note collections

  • Implement rate limiting for API calls
  • Regularly update Docker images
